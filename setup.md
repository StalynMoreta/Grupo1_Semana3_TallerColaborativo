# Guía de Instalación y Ejecución

Este proyecto utiliza Python y Jupyter Notebook. Sigue estos pasos para instalar las dependencias y ejecutar el notebook:

## 1. Clonar el repositorio

```bash
git clone https://github.com/StalynMoreta/Grupo1_Semana3_TallerColaborativo
cd Grupo1_Semana3_TallerColaborativo
```

## 2. Crear y activar un entorno virtual (opcional pero recomendado)

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## 3. Instalar las dependencias

```bash
pip install -r requirements.txt
```

## 4. Instalar Jupyter Notebook (si no está instalado)

```bash
pip install notebook
```

## 5. Ejecutar Jupyter Notebook

```bash
jupyter notebook
```

Luego, abre el archivo `Notebooks/Grupo1_semana3.ipynb` desde la interfaz de Jupyter.

---

**Notas:**
- Si tienes problemas con alguna dependencia, asegúrate de tener la versión de Python recomendada (por ejemplo, Python 3.8+).
- Para salir del entorno virtual, ejecuta `deactivate`.
- Si usas Anaconda, puedes crear un entorno con `conda` y luego instalar las dependencias.
